# Simple-Weather-Website
A dynamic weather web application that fetches real-time weather data using OpenWeather API with animated temperature-based UI themes and auto-adjusting background color
Features-:
1.Search weather by city name
2.Display temperature in Celsius
3.Show humidity and weather description
4.Weather condition emoji
5.Temperature-based background color changes
6.Loading spinner while fetching data
7.Error message shake for invalid city input

Technologies Used-
1.HTML
2.CSS(Flexbox,Animations and Gradients)
3.JavaScript(ES6)
4.Fetch API
5.OpenWeather API

Purpose

This project was created to practice:
-Working with APIs
-Async/Await in JavaScript
-DOM manipulation
-Conditional styling
-Improving UI with animations
